# Onion Web

A Next.js project scaffold with built-in internationalization.

## Setup

```bash
bun install
```

## Run

```bash
bun run dev
```

## Build

```bash
bun run build
```

## i18n

- Supported locales: `zh`, `en`
- Locale config: `src/i18n/config.ts`
- Locale messages: `src/i18n/messages.ts`
- Locale route: `src/app/[locale]/page.tsx`
